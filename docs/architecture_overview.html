Vibe Coding Workflow

1

Generate Plan
Run generate_bootstrap_prompt.py "intent" to synthesize a structured session prompt from your English description.

2

Start Session
Copy the generated prompt from bootstrap_prompts/ into a new Gemini session.

3

Execute
Lead Agent implements following Standing Instructions: read constitution, update changelog, run Bruno before commit.

4

Maintain
Run verify_structure.py and optimize_changelog.py after every feature to prevent context rot.
